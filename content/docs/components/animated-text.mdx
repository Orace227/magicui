## Props

| Prop        | Type               | Description                                                                              | Default         |
| ----------- | ------------------ | ---------------------------------------------------------------------------------------- | --------------- |
| `text`      | `string`           | The text to display and animate.                                                         | `"Hello world"` |
| `className` | `string`           | Additional classes to apply to the text element.                                         | `""`            |
| `delay`     | `number`           | The delay in milliseconds before the animation starts.                                   | `500`           |
| `fontSize`  | `string \| number` | The font size of the text, can be in any valid CSS unit or a number representing pixels. | `"2rem"`        |
